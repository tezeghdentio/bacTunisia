# 🇹🇳 Bac Tunisie — Sujets Théoriques et Pratiques

![GitHub repo size](https://img.shields.io/badge/Content-Bac%20Tunisie-green)

![Sections](https://img.shields.io/badge/Sections-All-orange)

Une collection organisée des **sujets du baccalauréat tunisien** pour **toutes les sections**, incluant :

- les **épreuves théoriques**
- les **épreuves pratiques**
- les **corrections** *(si disponibles)*
- les sujets classés **par section**, **par matière** et **par année**

Ce projet a pour objectif d’aider les élèves, enseignants et candidats libres à :

- réviser efficacement
- s’entraîner avec les anciens examens
- accéder rapidement aux sujets par section
- centraliser les ressources du Bac tunisien dans un seul espace

## 🎓 Sections disponibles

Le repository peut contenir les sections suivantes :

- **Mathématiques**
- **Sciences expérimentales**
- **Technique**
- **Économie et gestion**
- **Lettres**
- **Sport**

---

## 🗂 Organisation du repository

Les sujets sont organisés de manière simple et logique :

1. **Section**
2. **Matière**
3. **Année**
4. **Sujet / Correction**

Cette structure permet de retrouver rapidement un examen précis.
